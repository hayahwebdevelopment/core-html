Some of the List of Tags

html: This tag is used to indicate the start and end of an HTML document.

head: This tag is used to define the head section of an HTML document, which contains meta information, title elements, and links to external resources.

title: This tag is used to define the title of an HTML document, which appears in the browser's title bar and is used for SEO purposes.

body: This tag is used to define the body section of an HTML document, which contains the content that is displayed on the webpage.
